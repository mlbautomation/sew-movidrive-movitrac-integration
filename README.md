# SEW MOVIDRIVE / MOVITRAC – PROFINET Integration

Implementación y puesta en marcha de sistemas de accionamiento SEW (MOVIDRIVE y MOVITRAC) con integración PROFINET en Siemens TIA Portal, incluyendo parametrización, control de movimiento, feedback con encoder y diagnóstico.

---

## Objetivo

Desarrollar una integración completa del sistema de accionamiento:

- PLC Siemens ET200SP
- Variadores SEW (MOVIDRIVE / MOVITRAC)
- Motor DRN con encoder
- Comunicación PROFINET

Enfoque en puesta en marcha real.
